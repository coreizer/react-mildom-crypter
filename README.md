# mildom-livechat-react

<a href="https://www.mildom.com">Mildom (ミルダム) </a>のライブチャットの取得と表示

## 📦 ビルド方法

```bash
# 依存関係パッケージをインストールします
$ yarn install

# ボットをホットリロードで実行する
$ yarn dev

# ローカルサーバーの起動
$ yarn build
```
